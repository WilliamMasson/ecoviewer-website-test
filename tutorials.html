// Tutorials & Demo

Learn to use
EcoViewer

Step-by-step guides to help you get the most out of EcoViewer — from first launch to advanced ecological characterisation workflows.

// Quick Start

01

Launch the Application

Open EcoViewer by clicking "Launch App" in the navigation. The app runs entirely in your browser via Google Earth Engine — no installation required. You may be prompted to sign in with a Google account that has Earth Engine access.

02

Navigate the Map

Use the interactive map to zoom and pan to your area of interest. EcoViewer provides global coverage — from tropical forests to arctic tundra. Use the scroll wheel or trackpad to zoom, and click-drag to pan across the landscape.

03

Select Data Layers

Use the layer panel to toggle different environmental reference datasets. Each layer provides a different ecological lens — vegetation indices, land cover classifications, climate surfaces, and high-resolution imagery. Layers can be stacked and compared.

04

Inspect Pixel Values

Click anywhere on the map to retrieve data values for that location across all active layers. The inspector panel displays the underlying dataset values, providing precise ecological context for any point on Earth.

05

Export & Use Your Data

Use EcoViewer's reference data to inform your annotation workflows, characterise landscape types, or validate ecological classifications. Reference the data source documentation for citation and licensing information.
