Mission

Why we built
EcoViewer

Ecological characterisation at scale requires consistent, high-quality reference data. EcoViewer was built to bridge the gap between raw satellite imagery and meaningful ecological interpretation — providing practitioners and researchers with a unified, multi-source reference layer accessible anywhere in the world.

The GBIF integration adds a biodiversity dimension: species occurrence data is mapped directly to ecosystem types, enabling richer, species-informed landscape characterisation.
