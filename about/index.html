// About EcoViewer

The project behind
the reference library

EcoViewer is an open-access environmental reference platform built to support ecological fieldwork, remote sensing, and conservation science — globally.

// What we built

Bridging data and
ecological practice

The challenge for many ecologists and remote sensing practitioners isn't a lack of data — it's that environmental datasets are scattered across dozens of repositories, each with different formats, resolutions, and coverage gaps. EcoViewer was built to solve that.

By integrating multiple high-resolution datasets into a single, coherent reference layer — and surfacing it through a fast, browser-based interface powered by Google Earth Engine — EcoViewer gives analysts the ecological context they need to characterise landscapes, annotate satellite imagery, and conduct reproducible comparative studies at any scale.

01

Open Access

All reference layers are freely accessible, with no login or licensing barriers for scientific and conservation use.

02

Multi-Source Integration

Data is aggregated from verified environmental repositories and harmonised into a consistent, queryable reference layer.

03

Planetary Scale

Powered by Google Earth Engine, EcoViewer provides high-resolution coverage across all biomes and continents worldwide.

04

Built for Practice

Designed with ecologists and remote sensing specialists in mind — not just data engineers — for real-world annotation and fieldwork support.
